# Agrolink - Ūkininkų Platforma

React Native programėlė ūkininkams, veikianti Android ir iOS.

## Funkcijos

- **Technikos turgus** - Žemės ūkio technikos pardavimas ir pirkimas
- **Ūkininkų paslaugos** - Paslaugų užsakymas (kultivavimas, sėja ir kt.)
- **Naujienlaiškis** - Aktualios naujienos ūkininkams
- **Kalendorius** - Ūkio darbų planavimas

## Projekto struktūra

- `App.js` - Pagrindinis navigacijos failas
- `src/screens/` - Visi ekranai
- `src/database/` - SQLite duomenų bazė

## Technologijos

- React Native
- Expo
- SQLite
- React Navigation
- Bottom Tabs Navigation

## Kaip paleisti

```bash
npm install
npm start
```

Tada paspauskite:
- `a` - Android emuliatorius
- `i` - iOS simuliatorius
- Arba nuskenuokite QR kodą Expo Go app
